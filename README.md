# jadx-bridge

Atomic plugin exposing only `jadx_bridge`.

## Tool

- `jadx_bridge`

## Dependencies

- Jadx CLI (`jadx`)

## Environment

- `PLUGIN_JADX_BRIDGE_ENABLED` / `PLUGINS_JADX_BRIDGE_ENABLED`

## Notes

- Decompile output defaults to `artifacts/jadx-decompile/` under current working directory.
